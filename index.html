Visual Studio is an online and offline code editor which can be used for HTML, CSS, and Javascript

A console is the part of a computer used for communicating between the user and the system. It is used to allow a user to input commands and receive visual output from a computer or computer system.

To open the console, you have to right click and press inspect. and go to console
